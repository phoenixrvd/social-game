# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, do NOT open a public issue.

Please use GitHub's private vulnerability reporting feature to report the issue confidentially.

## Supported Versions

Only the latest release on the `main` branch is supported.

Older releases are not maintained and will not receive security patches.

## Disclaimer

This project is experimental and provided "as is", without warranty of any kind.

Use at your own risk.
